# Shree Varaha | Design and Construction

A professional, high-performance web presence for **Shree Varaha Design and Construction**. This project has been refined for a clean, formal aesthetic suitable for high-level government and industrial projects.

## 🏗️ Project Overview

Shree Varaha is committed to setting benchmarks in the design and construction industry. This website showcases our precision, professionalism, and commitment to exceeding client expectations.

### Key Features
- **Modern Static Design**: Professional navy-blue accents and a clean, traditional layout.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile viewing.
- **Service Showcases**: Detailed overviews of Town Planning, Construction Management, Structural Engineering, and more.
- **Project Gallery**: High-quality visualization of completed and ongoing projects.

## 🚀 Technologies Used
- **HTML5 & Vanilla CSS**: For lightweight, high-performance rendering.
- **JavaScript**: Smooth scroll reveals and interactive magnetic elements.
- **Vite**: Modern frontend build tool for lightning-fast development and optimized production builds.

## 🛠️ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (latest LTS recommended)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/preethamamin22/shreevaraha.git
   ```
2. Navigate to the project directory:
   ```bash
   cd shreevaraha
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
To start the local development server:
```bash
npm run dev
```

### Production Build
To create an optimized production bundle:
```bash
npm run build
```
